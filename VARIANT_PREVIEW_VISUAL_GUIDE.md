## Interaction Flow

```
User lands on product page
        ↓
Sees product details + "🎁 View All X Variants" button
        ↓
(Optional) Clicks to expand preview
        ↓
Reviews all variants: prices, colors, attributes, stock
        ↓
Closes preview (or keeps it open)
        ↓
Selects specific variant using attribute selectors
        ↓
Sets quantity
        ↓
Clicks "Add To Cart"
        ↓
Item added with selected variant details
```

## Complete User Story

**Scenario**: Customer wants to buy a personalized mug

**Before** (Without preview):
- "Which mug should I get?"
- Blindly selects color
- Adds to cart
- Later realizes they wanted a different size
- Has to edit cart or remove/re-add

**After** (With preview):
- Sees "View All 8 Variants"
- Clicks to see all options with prices
- Compares Red M (₹499), Red L (₹549), Blue M (₹499)
- Decides: "I want the Red Large one!"
- Selects Color: Red, Size: L
- Adds to cart with confidence
- Checkout smooth, correct variant selected
